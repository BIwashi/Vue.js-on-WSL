# Vue.js-on-WSL
Install Android SDK and Vue.js on Windows Bash (WSL)

`setup`実行後，username以下のディレクトリに作成した共有ディレクトリに，解凍したandroid sdkのtoolsを置いたのち，`setup2.sh`を実行
